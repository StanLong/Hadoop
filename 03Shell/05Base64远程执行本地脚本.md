# Base64 远程执行本地脚本

```shell
[hadoop@hadoop101 ~]$ base64 -w0 test.sh 
aG9zdG5hbWUKZWNobyBMb24gR3JlYXQgQ2hpbmEK
[hadoop@hadoop101 ~]$ echo `base64 -w0 test.sh` | base64 -d | bash
hadoop101
Long Great China
[hadoop@hadoop101 ~]$ ssh 192.168.6.106 "echo `base64 -w0 test.sh` | base64 -d | bash"
hadoop106
Long Great China
```
