# 扫描本地网络中存活的机器

**案例需求：**判断本地网络中哪些IP被使用

**案例分析：**采用ping的方式判断IP是否被占用

```
a、能ping通说明占用 b、不能ping通说明未被占用
```

b、命令

```
ping -c1 IP # -c1 ping 一次
```

**算法：**

1、ping ip

2、分析ping结果

3、输出结果

```shell
#!/bin/bash
#  () 中的代码段可以分组，表示这段代码可以并行执行， & 表示后台执行
netsub="192.168.235."
for ip in `seq 1 255`
do
(
        ping -c1 $netsub$ip &>/dev/null
        if [ $? -eq 0 ];then
                echo "$netsub$ip 正常"
        else
                echo "$netsub$ip 掉线"
        fi
) &
done
```
